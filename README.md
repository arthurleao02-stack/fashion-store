# Fashion Store 👕👗

E-commerce de roupas online moderno, desenvolvido com tecnologias de ponta para oferecer uma experiência de compra fluida e segura.

## 🚀 Tecnologias

- **Frontend:** [Next.js](https://nextjs.org/), [React](https://reactjs.org/), [Tailwind CSS](https://tailwindcss.com/)
- **Backend:** [Node.js](https://nodejs.org/), [TypeScript](https://www.typescriptlang.org/)
- **Banco de Dados:** [PostgreSQL](https://www.postgresql.org/)
- **Integração de Pagamento:** [Mercado Pago](https://www.mercadopago.com.br/developers/)
- **Infraestrutura:** [Docker](https://www.docker.com/)

## 🛠️ Estrutura do Projeto

O repositório está dividido em:

- `/frontend`: Aplicação cliente desenvolvida em Next.js.
- `/backend`: API RESTful desenvolvida em Node.js com TypeScript.
- `/docs`: Documentação adicional da API e guias de configuração.

## ⚙️ Como Executar

Consulte os seguintes guias para configurar o ambiente:

1. [Guia de Instalação](docs/SETUP.md)
2. [Guia de Desenvolvimento](DEVELOPMENT.md)

## 📦 Docker

Você pode subir o ambiente completo utilizando o Docker Compose:

```bash
docker-compose up -d
```
